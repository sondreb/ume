You can create any number of invites to communities you own.

You have to be online for members to activate their invites.

Invites are only stored locally on this device, if you run "Clear Device" and restore on another, the invites will no longer be valid.

Invites are single-use.

Invites can include information about gateway. This can be used to simplify the experience for new members, but it exposes the gateway used by the community. Do not include the gateway in the invitation for additional secure, and communicate the gateway with the member through other secure channels.
