Privacy and Security

üme was developed from the need of a secure and privacy focused community software, that did not rely on tedious installation and setup of servers and databases.

Most community software today, requires a centralized server and database. This comes with many costs, and most importantly, it's detrimental to your privacy and security.

We have written extensively on how privacy and security is ensured with üme. Two main points before you read more:

üme runs in your (modern) web browser, and üme relies on end-to-end encryption.

Main Features

  • Open Source

    üme is open source and relies on standards for encryption and secure communication.

  • User Hosted

    üme only stores data on each users device. There are no central storage.

  • Shiny & Modern

    üme relies on the latest web standards and technologies in modern browsers.
